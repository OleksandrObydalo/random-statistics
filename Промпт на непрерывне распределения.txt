Сохраняя работоспособность предыдущей версии, добавь возможность выбирать типы непрерывных распределений. Это задание усложнено тем, что для разных распределений нужно вводить разные параметры.
Из-за того что непрерывные случайные величины имеют бесконечное количество возможных значений, не имеет смысла считать вероятность конкретного значения P(X = k).
Вместо этого можно рассчитать вероятность нахождения случайной величины в определенном интервале [a, b]. Тоесть P(a >= X >= b).
Также для каждого такого распределения сущестует функция плотности: p(x) = P(x >= X >= x + dx)/dx где dx стремиться к нулю.
Для рассчета вероятности находжения случайной величины в интервале, используеться кумулятивная функция, которая являеться первообразной от функциии плотности. F(x) = integral(0, x, p(x)dx) Тоесть интеграл от 0 до x по p(x).
P(a >= X >= b) = F(b) - F(a) = integral(a, b, p(x)dx)
Виды распределений: Равномерное непрерывное: от дробное число минимума до максимума (параметры). Экспоненциальное рапределение - непрерывный аналог Пуассоновского распределения. функция плотности: p(x) = lambda * (e^(lambda * x)) Где lambda это параметр распределения. 
Нормальное оаспределение - непрерывный аналог биномиального. Функция плотности: p(x) = ( e^(-((x - mean)^2)/(2 * sigma * sigma)) ) / (sqrt(2 * pi) * sigma). Где mean - среднее значение (параметр), sigma - диспресия (параметр).
